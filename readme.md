# Python cryptography Build

A project for building the following packages:

 - cryptography
 - pyopenssl

along with:

 - OpenSSL

for the operating systems:

 - Windows x86
 - Windows x64
 - OS X x64
 - Linux i686
 - Linux x86_64

and the following versions of Python:

 - 2.6
 - 3.3

## Downloads

See https://github.com/codexns/pyopenssl-build/releases for downloads.
